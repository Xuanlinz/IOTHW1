## Completed Tasks

### 1. Web Development & Design
* **Structural Architecture (index.html):** Developed a clean, single-page layout featuring a dynamic greeting, professional biography, and dedicated sections for technical skills (e.g., Digital IC Design, EDA tools) and IoT project showcases.
* **Visual Styling (style.css):** Implemented a "Glassmorphism" design system utilizing CSS variables, blurred glass effects, and smooth gradients for a high-end, modern aesthetic.
* **Responsive Features:** Integrated JavaScript for real-time clock updates and ensured fluid responsiveness across mobile, tablet, and desktop environments.

### 2. Git & GitHub Integration
* **Version Control:** Initialized a local Git repository and established a standardized commit workflow to maintain code integrity and history.
* **GitHub Pages Deployment:** Successfully configured the repository for automated hosting via GitHub Pages, enabling public access to the live site.
* **Privacy Protection:** Configured the project to utilize GitHub’s `noreply` email system to ensure personal contact data is protected within the public commit history.
