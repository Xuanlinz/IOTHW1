# Project Summary - March 3, 2026

本專案為 **Xuan-Lin** 開發的個人化網頁，旨在作為物聯網課程（IoT）的學習成果展示。本專案透過整合前端網頁開發技術與 **GitHub Pages** 自動化部署流程，建立了一個具有專業感且符合電機工程背景的互動式介面。

## Completed Tasks

### 1. 前端開發與設計 (Web Development)
* **架構設計 (index.html):** 建立了一個結構清晰的單頁式（Single-page）佈局，包含個人簡介、電機專業技能（如：Digital IC、EDA 工具）以及 IoT 專案展示區。
* **視覺風格 (style.css):** 採用了現代化的 **Glassmorphism（玻璃擬態）** 風格，結合半透明層次、流暢的漸層背景以及微動效（Micro-animations），提升使用者介面的專業質感。
* **互動功能:** 加入了即時時間顯示與響應式導覽功能，確保網頁在不同裝置（如手機、平板與桌機）上皆能完美呈現。

### 2. Git 版本控制與雲端部署 (Integration & Deployment)
* **版本控制管理:** 於本地端初始化 Git 儲存庫，並建立標準化的 Commit 流程，確保程式碼變更紀錄的可追溯性。
* **GitHub Pages 部署:** 成功透過 GitHub Settings 完成靜態部署，將程式碼推送到遠端儲存庫並自動完成網頁託管。
* **隱私與安全性設置:** 配置了 GitHub 的專屬 `noreply` 電子郵件地址，在提交程式碼的同時確保個人隱私資訊不外洩。
